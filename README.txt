# Theatre1 Static Website

## Structure:
- index.html (main site file)
- /assets/ (folder with all images: show posters, founder headshots, logos)

## How to View Locally:
1. Download and unzip the folder.
2. Open `index.html` in a web browser (Chrome, Firefox, Edge, etc.)

## How to Deploy:
You can upload the folder to:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting service.

No server-side code needed. Fully static!

## Notes:
- Past Performances posters are in /assets/
- Founder headshots are in /assets/ (JPG format, circular)
- TailwindCSS is used via CDN
